GPU e Placa-mãe:
paralelismo e o esqueleto

A GPU é o processador especializado em fazer milhares de coisas ao mesmo tempo — por isso domina gráficos e inteligência artificial. A placa-mãe é o esqueleto que conecta todos os componentes. Entenda o que cada uma faz e por que elas importam para quem programa.

⏱ ~20 min
📶 Iniciante
Módulo 1 de 5

O que é a GPU?

GPU significa Graphics Processing Unit — Unidade de Processamento Gráfico. Ela nasceu para renderizar gráficos em jogos e aplicações visuais, mas hoje é usada em muito mais do que isso: inteligência artificial, machine learning, simulações científicas, criptomoedas e vídeo.

A GPU é um processador como a CPU, mas com uma filosofia completamente diferente. Enquanto a CPU tem poucos núcleos extremamente poderosos (otimizados para executar tarefas complexas em sequência), a GPU tem milhares de núcleos menores (otimizados para executar a mesma tarefa em muitos dados ao mesmo tempo).

🔢
Números reais: Uma CPU moderna de alto desempenho tem entre 8 e 24 núcleos. Uma GPU de médio porte como a NVIDIA RTX 4060 tem 3.072 núcleos CUDA. A RTX 4090 tem 16.384 núcleos. São ordens de magnitude diferentes.

CPU vs GPU: serial vs paralelo

A diferença entre CPU e GPU é uma diferença de filosofia de design:

CPU — sequencial

Poucos núcleos, muito poder

Ótima para tarefas que dependem do resultado anterior: lógica de negócio, condicionais complexos, código com muitas dependências entre passos. Um chef Michelin — faz poucas coisas, mas com maestria.

GPU — paralelo

Milhares de núcleos simples

Ótima para tarefas independentes e repetitivas: calcular a cor de cada pixel, aplicar a mesma operação a um array de milhões de números. Mil funcionários de fast food — simples, mas em massa.

🎮
Por que jogos usam GPU? Uma tela Full HD tem 1.920 × 1.080 = 2.073.600 pixels. A cada frame (60 vezes por segundo em um jogo), a GPU precisa calcular a cor de cada um desses pixels, considerando iluminação, sombras, reflexos e texturas. São mais de 124 milhões de cálculos por segundo — e todos eles são independentes entre si. Isso é exatamente o que a GPU faz melhor.

Por que a IA adora GPUs?

Quando você ouve que treinar um modelo de IA leva horas em GPU e meses em CPU, a razão é matemática. O coração do machine learning é a multiplicação de matrizes — operações onde você multiplica tabelas gigantescas de números.

Uma matriz de 1000×1000 multiplicada por outra de 1000×1000 requer 1 bilhão de multiplicações. Uma CPU faz isso em sequência — núcleo por núcleo. Uma GPU pode fazer milhares dessas multiplicações em paralelo, reduzindo o tempo dramaticamente.

  1. →
    Treinamento de redes neurais É basicamente multiplicar matrizes bilhões de vezes, ajustando parâmetros a cada rodada. GPUs fazem isso em horas; CPUs levariam meses.
  2. →
    Inferência (usar o modelo) Quando você usa o ChatGPT, a resposta é gerada por GPUs em data centers — aplicando o modelo treinado ao seu texto em tempo real.
  3. →
    CUDA e ROCm NVIDIA criou o CUDA — uma plataforma que permite programar GPUs para tarefas gerais, não só gráficos. É por isso que NVIDIA domina o mercado de IA hoje.

GPU dedicada vs. integrada

GPU integrada

Dentro do mesmo chip da CPU

A maioria dos processadores modernos (Intel Core, AMD Ryzen, Apple M3) tem uma GPU integrada. Usa a RAM do sistema. Suficiente para uso cotidiano, mas fraca para jogos pesados ou IA.

GPU dedicada

Placa de vídeo separada

Um componente à parte com sua própria memória (VRAM). NVIDIA RTX ou AMD Radeon. Necessária para jogos modernos, edição de vídeo profissional, machine learning.

💡
Para o curso: Você não precisa de GPU dedicada para aprender a programar — nem para as 5 fases deste curso. GPU dedicada começa a importar quando você chegar em machine learning ou desenvolvimento de jogos 3D. GPU integrada é mais do que suficiente para Python, web, algoritmos e projetos autorais.

Placa-mãe: o esqueleto do computador

A placa-mãe (motherboard) é o componente que conecta tudo. Ela é uma placa de circuito impresso com slots, conectores e chips que permitem que a CPU, RAM, SSD, GPU e periféricos se comuniquem entre si.

Sem a placa-mãe, os componentes são apenas peças isoladas. É ela que define o barramento (os "corredores" por onde os dados trafegam), os slots disponíveis e a compatibilidade entre componentes.

Soquete da CPU

Onde a CPU encaixa

Cada geração de CPU tem um soquete diferente. LGA1700 (Intel 12ª/13ª gen), AM5 (AMD Ryzen 7000). Você não pode colocar uma CPU numa placa com soquete incompatível.

Slots de RAM

Onde os pentes encaixam

Normalmente 2 ou 4 slots. A maioria das placas suporta até 64 GB ou 128 GB. O tipo de RAM (DDR4 ou DDR5) precisa ser compatível com a placa-mãe.

Slots PCIe

Para GPU e SSDs NVMe

PCIe x16 é onde a GPU encaixa. Slots M.2 (para SSDs NVMe) também usam PCIe. A geração (PCIe 4.0, 5.0) determina a largura de banda disponível.

Chipset

O gerente do tráfego

O chipset é um chip na placa-mãe que gerencia a comunicação entre CPU, memória e periféricos. Determina quantas USBs, quantos slots M.2 e quais recursos estão disponíveis.

BIOS / UEFI: o primeiro software

Antes do sistema operacional carregar, quem está no controle é o BIOS (Basic Input/Output System) ou seu sucessor moderno, o UEFI (Unified Extensible Firmware Interface). É um firmware gravado num chip na placa-mãe que inicializa todos os componentes de hardware e passa o controle para o sistema operacional.

É no BIOS/UEFI que você configura a ordem de boot (de qual dispositivo o computador tenta iniciar primeiro), ativa ou desativa componentes, e pode ver as temperaturas dos componentes. Todo desenvolvedor deveria saber acessar o BIOS ao menos uma vez — geralmente pressionando Delete, F2 ou F10 logo ao ligar o computador.

Termos-chave desta aula

GPU Graphics Processing Unit. Processador com milhares de núcleos para operações paralelas. Essencial para gráficos e IA.
CUDA Plataforma da NVIDIA para programar GPUs em tarefas gerais (não só gráficos). Dominante em machine learning.
VRAM Video RAM. Memória exclusiva da GPU dedicada. Mais VRAM = texturas maiores, modelos de IA maiores.
GPU integrada GPU embutida no mesmo chip da CPU. Usa a RAM do sistema. Suficiente para uso geral e programação.
Placa-mãe Placa de circuito que conecta todos os componentes. Define compatibilidade e recursos disponíveis.
Chipset Chip da placa-mãe que gerencia comunicação entre componentes. Determina número de portas e slots.
PCIe Peripheral Component Interconnect Express. Barramento de alta velocidade para GPU, SSDs NVMe e placas de expansão.
BIOS / UEFI Firmware da placa-mãe. Inicializa o hardware antes do sistema operacional e permite configurações de baixo nível.
