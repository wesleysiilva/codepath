Fase 1 · Linha de Comando

Navegação:
mover-se pelo sistema de arquivos

pwd, ls, cd, mkdir, rm — são os comandos que você vai usar todos os dias. Com eles você navega pastas, lista arquivos, cria estruturas de projeto e apaga o que não precisa. Tudo sem tirar as mãos do teclado.

⏱ ~15 min
📶 Iniciante
Módulo 5 de 5

Referência rápida dos comandos essenciais

pwd
Windows: cd (sem args)
Print Working Directory. Mostra o caminho absoluto do diretório atual. Sempre use quando estiver perdido.
ls / ls -la
Windows: dir
List. Lista arquivos e pastas. -l formato longo (permissões, tamanho, data). -a mostra arquivos ocultos (que começam com .).
cd <caminho>
Windows: cd <caminho>
Change Directory. Navega para outra pasta. cd .. sobe um nível. cd ~ vai para a home. cd - volta ao diretório anterior.
mkdir <nome>
Windows: mkdir <nome>
Make Directory. Cria uma pasta. mkdir -p a/b/c cria a hierarquia completa de uma vez.
rm <arquivo>
Windows: del <arquivo>
Remove arquivo. rm -r pasta/ remove pasta e todo o conteúdo. Não vai para a lixeira! Irreversível.
rmdir <pasta>
Windows: rmdir <pasta>
Remove pasta vazia. Se a pasta tiver conteúdo, use rm -r. Mais seguro que rm -r para evitar acidentes.

Na prática: criando um projeto

bash
# Onde estou?
$ pwd
/home/wesley

# O que tem aqui?
$ ls
Desktop Documents Downloads projetos

# Criar estrutura do projeto
$ mkdir -p projetos/meu-app/src
$ cd projetos/meu-app
~/projetos/meu-app$ ls -la
total 8
drwxr-xr-x 3 wesley staff 96 Apr 19 10:00 .
drwxr-xr-x 5 wesley staff 160 Apr 19 10:00 ..
drwxr-xr-x 2 wesley staff 64 Apr 19 10:00 src

# Voltar para home
~/projetos/meu-app$ cd ~
~$ pwd
/home/wesley

Caminhos absolutos e relativos no terminal

Todo comando que aceita um caminho funciona com absoluto ou relativo. Escolha o mais conveniente para a situação:

bash — exemplos de caminho
# Estou em: /home/wesley/projetos

# Caminho relativo (a partir de onde estou)
$ cd meu-app/src

# Caminho absoluto (sempre funciona)
$ cd /home/wesley/projetos/meu-app/src

# Subir um nível (..)
$ cd ..
# Agora em: /home/wesley/projetos/meu-app

# Subir dois níveis
$ cd ../..
# Agora em: /home/wesley

Atalhos de teclado essenciais

Tab

Autocompletar

O atalho mais importante do terminal. Digite as primeiras letras de um comando ou caminho e aperte Tab. O shell completa automaticamente. Aperte Tab duas vezes para ver todas as opções.

↑ / ↓

Histórico de comandos

Seta para cima navega pelo histórico de comandos. Seta para baixo volta ao mais recente. history mostra os últimos N comandos. Ctrl+R busca no histórico.

Ctrl+C

Cancelar comando

Interrompe o comando em execução. Se um programa travou ou está rodando mais tempo que o esperado, Ctrl+C o cancela imediatamente. Ctrl+Z o pausa (coloca em background).

Ctrl+L

Limpar tela

Equivalente a clear. Limpa o terminal sem apagar o histórico. O scroll ainda pode acessar o que estava antes. Mantém o prompt no topo da janela.

Termos-chave desta aula

pwd Print Working Directory. Exibe o caminho absoluto do diretório atual. Use sempre que não souber onde está.
ls List. Lista arquivos e diretórios. -l = formato longo, -a = inclui ocultos, -la = ambos.
cd Change Directory. Muda o diretório de trabalho. cd .. sobe um nível, cd ~ vai para home, cd - volta ao anterior.
mkdir -p Make Directory com flag -p (parents). Cria diretórios aninhados de uma vez: mkdir -p a/b/c cria a, b e c.
rm -r Remove recursivamente pasta e todo o conteúdo. Irreversível — não vai para lixeira. Use com cuidado.
Tab (autocomplete) Atalho mais útil do terminal. Completa comandos, caminhos e nomes de arquivo automaticamente. Dois Tabs mostra todas as opções.
